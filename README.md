# Python Web Scraper
A simple Python script that scrapes data from a website and saves the results locally.

# What it does
- Sends HTTP requests to a target webpage
- Parses HTML content using BeautifulSoup
- Extracts specific data (titles, links, prices, etc.)
- Saves the output to a CSV file

# Tech Stack
- Python 3.9+
- requests
- beautifulsoup4

# Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/AparnaAmonkar22/Python.git

cd Python/WebScrapping
pip3 install -r requirements.txt

Run the scraper:
python WebScrapping.py

Scraped data will be saved to:
Top_100_US_Companies.csv
